# Aprendiendo SQL

Repositorio para practicar y documentar mi aprendizaje de SQL desde cero.

## Ejercicio 1 - Tienda de libros

Un ejercicio básico que simula la base de datos de una tienda de libros con autores y títulos.

Cubre:

- Creación de tablas con `CREATE TABLE`
- Inserción de datos con `INSERT`
- Consultas básicas con `SELECT`, `WHERE` y `ORDER BY`
- Combinación de tablas con `JOIN`
- Agrupación con `GROUP BY` y funciones como `COUNT`, `AVG`, `SUM`
- Actualización de datos con `UPDATE`

## Cómo probarlo

El archivo se puede ejecutar directo en el navegador sin instalar nada usando [SQLite Online](https://sqliteonline.com). Solo pegás el contenido y le das Run.

## Motor usado

SQLite (compatible con el estándar SQL básico)
